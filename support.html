Support and Frequently Asked Questions

What does the Add-on do?

Data Toolkit for Sheets provides tools for enhanced data analysis within Google Sheets, including API integration and AI assistance, to help you perform data-related tasks more efficiently.

How do I install the Add-on?

The Add-on can be easily installed from the Google Workspace Marketplace. Simply search for "Data Toolkit for Sheets" and follow the installation instructions.
