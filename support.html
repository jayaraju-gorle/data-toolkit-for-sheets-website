Support and Frequently Asked Questions

Find answers to common questions and learn how to get the most out of Data Toolkit for Sheets.

What does the Add-on do?

Data Toolkit for Sheets provides tools for enhanced data analysis within Google Sheets, including API integration and AI assistance, to help you perform data-related tasks more efficiently.

How do I install the Add-on?

The Add-on can be easily installed from the Google Workspace Marketplace. Simply search for "Data Toolkit for Sheets" and follow the installation instructions.

  1. Open Google Sheets
  2. Click on "Extensions" in the menu
  3. Select "Add-ons" and then "Get add-ons"
  4. Search for "Data Toolkit for Sheets"
  5. Click "Install"

Common Issues

The add-on is not showing up after installation

If the add-on is not appearing:

  1. Refresh your Google Sheets page
  2. Check that you're signed in with the same Google account used for installation
  3. Look under the "Extensions" menu for "Data Toolkit for Sheets"
  4. Try closing and reopening Google Sheets

API connection issues

If you're experiencing API connection problems:

  1. Verify your API key is entered correctly
  2. Check your internet connection
  3. Ensure the API endpoint is correct and accessible
  4. Look for any quota limitations on the API service you're connecting to

Performance optimization

To improve performance when working with large datasets:

  1. Limit the range of cells being processed
  2. Use batch operations instead of cell-by-cell processing when possible
  3. Close unused tabs and extensions in your browser
  4. Consider splitting very large datasets into separate sheets
