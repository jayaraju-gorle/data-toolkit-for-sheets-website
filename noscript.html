How to Enable JavaScript

Chrome

1. Click the three dots in the top right corner
2. Go to Settings > Privacy and security > Site Settings > JavaScript
3. Make sure "Sites can use JavaScript" is enabled

Firefox

1. Click the three lines in the top right corner
2. Go to Settings > Privacy & Security
3. Scroll to JavaScript and make sure it's enabled

Safari

1. Go to Safari > Preferences > Security
2. Make sure "Enable JavaScript" is checked
